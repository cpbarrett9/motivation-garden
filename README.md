<h1>Overview</h1>

Motivation Garden is a game-ified productivity app that rewards completed tasks with items for a virtual garden. Tasks can be created, managed, and marked as complete. Completing a task rewards 20 coins which can be spent on animals and plants for the garden, visualizing your progress.
